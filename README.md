# Innovati assignment
Reto de desarrollo NodeJS + VueJS + Google Cloud

## Prerequisites
- Node
- express
- morgan
- mysql
- webpack
- dotenv
- vue
- Bootstrap 4

## Run + deployment

- Locally:

edit .env file
```
NODE_ENV=test
```
```
npm install
npm run dev
```


- Google Cloud -> [link](http://innovati-2.appspot.com/)
edit .env file
```
NODE_ENV=production
```
```
npm install
gcloud app deploy
gcloud app browse
```
